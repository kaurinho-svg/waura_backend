# Общий список команд Telegram-бота Waura

В этом документе перечислены все команды, которые поддерживаются ботом, разделенные по уровням доступа пользователей.

## 👤 Для покупателей (Клиентов магазина)
Эти команды доступны всем обычным пользователям, которые зашли в бота магазина.

*   **/start** — Запустить бота. Открывает главное меню покупателя (Каталог, Поиск по категориям, AI-Стилист).

---

## 👔 Для владельцев и администраторов магазинов
Эти команды доступны только тем пользователям, чей Telegram ID указан как владелец магазина в базе данных (`telegram_id`), либо добавлен в список дополнительных администраторов (`admin_ids`).

*   **/admin** — Открыть панель управления магазином. Позволяет добавлять/удалять товары, категории и просматривать список заказов.
*   **/stats** — Посмотреть подробную статистику магазина: количество уникальных покупателей, активных товаров, заказов, общую выручку, а также остаток генераций для виртуальной примерки.
*   **/broadcast** — Запустить массовую рассылку (текст, фото, видео) по всем покупателям, которые когда-либо писали боту этого магазина (Доступно только при наличии **Premium** подписки).
*   **/listadmins** — Показать список всех текущих администраторов магазина.

**Только для главного владельца магазина:**
*   **/addadmin <Telegram ID>** — Выдать права администратора другому сотруднику. Позволяет ему пользоваться командами `/admin` и `/broadcast`. (Пример: `/addadmin 123456789`)
*   **/removeadmin <Telegram ID>** — Забрать права администратора у сотрудника.

---

## 👑 Для Суперадминистраторов (Владельцев платформы Waura)
Эти команды доступны только системным администраторам, чьи ID прописаны в файле `telegram_bot/config.py` в переменной `SUPER_ADMIN_IDS`.

*   **/addgen <ID Владельца> <Количество>** — Начислить лимиты (кредиты) на использование Виртуальной Примерки конкретному магазину. Указывается Telegram ID создателя/владельца магазина. (Пример: `/addgen 123456789 50`)
